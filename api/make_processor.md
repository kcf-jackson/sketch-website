## Make a handle to process header

### Description

Make a handle to process header

### Usage

    make_processor(pred, fun)

### Arguments

|        |                                                                 |
|--------|-----------------------------------------------------------------|
| `pred` | A function, taking a string and returning a logical.            |
| `fun`  | A function, taking a string and returning a 'shiny.tag' object. |

### Value

A header processor / handler.
