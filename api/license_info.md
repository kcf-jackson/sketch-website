## License information

### Description

License information

### Usage

    license_info(x)

### Arguments

|     |                                                   |
|-----|---------------------------------------------------|
| `x` | A character string; name of the library / assets. |

### Value

A named list containing the license information and the link from which
the information is extracted.

### Examples

    license_info("mathjs")
    license_info("p5")
