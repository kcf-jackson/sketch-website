## Empty functions

### Description

These functions do nothing. It is created to ensure the keywords 'let'
and 'declare' are defined.

### Usage

    let(...)

    declare(...)

    const(...)

### Arguments

|       |               |
|-------|---------------|
| `...` | Any arguments |

### Examples

    let (x)
    let (x = 1, y = 2)
    declare (x1, x2, x3)
