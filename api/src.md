## Get the source link of a JavaScript library

### Description

Get the source link of a JavaScript library

### Usage

    src(x)

### Arguments

|     |                                                    |
|-----|----------------------------------------------------|
| `x` | A character string; name of the JavaScript library |

### Value

A character string; the path to the library.

### Examples

    src("mathjs")
    src("p5")
